# Listify a 1-row data frame and assign the qcthat_SingleIssueTestResults class

Listify a 1-row data frame and assign the qcthat_SingleIssueTestResults
class

## Usage

``` r
AsSingleIssueTestResults(x)
```

## Arguments

- x:

  (`data.frame` or `list`) Object to convert to a list.

## Value

A `qcthat_Milestone` object, which is a list with elements `"Milestone"`
and `"IssueTestResults"`.

# Add an s to a word based on count

Add an s to a word based on count

## Usage

``` r
SimplePluralize(strSingular, intN)
```

## Arguments

- strSingular:

  (`character`) Singular form of the word.

- intN:

  (`integer`) Count to determine singular vs. plural.

## Value

A character string with the singular or plural form of the word.

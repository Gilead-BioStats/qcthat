# Git blame a file (raw)

Git blame a file (raw)

## Usage

``` r
BlameFileRaw(strFilePath)
```

## Arguments

- strFilePath:

  (`length-1 character`) A file path.

## Value

The result of
[`git2r::blame()`](https://docs.ropensci.org/git2r/reference/blame.html).
This function exists to make mocking easier.

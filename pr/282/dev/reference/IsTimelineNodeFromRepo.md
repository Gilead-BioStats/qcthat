# Check whether a single timeline node originates from the target repository

Check whether a single timeline node originates from the target
repository

## Usage

``` r
IsTimelineNodeFromRepo(lNode, strNameWithOwner)
```

## Arguments

- lNode:

  (`list`) A single timeline item node.

- strNameWithOwner:

  (`character(1)`) `"owner/repo"` of the target repo.

## Value

A length-1 `logical`.

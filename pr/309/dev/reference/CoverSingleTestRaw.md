# Run covr::environment_coverage (raw wrapper for mocking)

Run covr::environment_coverage (raw wrapper for mocking)

## Usage

``` r
CoverSingleTestRaw(envPkg, strTempFile)
```

## Arguments

- envPkg:

  (`environment`) Loaded package environment.

- strTempFile:

  (`length-1 character`) Path to a temporary test file.

## Value

A `coverage` object.

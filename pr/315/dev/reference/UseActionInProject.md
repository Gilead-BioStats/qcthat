# Use a GitHub Action in a project

Use a GitHub Action in a project

## Usage

``` r
UseActionInProject(strActionFilename, strPkgRoot = ".")
```

## Arguments

- strActionFilename:

  (`length-1 character`) The filename of the GitHub Action YAML file.

- strPkgRoot:

  (`length-1 character`) The root directory of the package.

## Value

(`length-1 character`) The URL of the GitHub Action YAML file that was
added.

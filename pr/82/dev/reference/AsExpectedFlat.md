# Listify a 1-row data frame and assign a class

Listify a 1-row data frame and assign a class

## Usage

``` r
AsExpectedFlat(x, lShape, chrClass)
```

## Arguments

- x:

  (`data.frame` or `list`) Object to convert to a list.

- lShape:

  (`list`) List with the expected shape.

- chrClass:

  (`character`) Class name(s) to assign to the object.

## Value

A list with the expected shape and class.
